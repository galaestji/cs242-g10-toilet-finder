# cs242-g10-toilet-finder
# ระบบค้นหาห้องน้ำใกล้ฉันในมหาวิทยาลัย

#ภาพรวมโครงงาน
ระบบนี้เป็นเว็บแอปพลิเคชันสำหรับช่วยนักศึกษา บุคลากร และบุคคลภายนอกค้นหาห้องน้ำที่ใกล้ที่สุดในมหาวิทยาลัย โดยเฉพาะเมื่ออยู่ในอาคารหรือพื้นที่ที่ไม่คุ้นเคย

ระบบจะใช้ตำแหน่งปัจจุบันของผู้ใช้ในการคำนวณและแสดงห้องน้ำที่ใกล้ที่สุด 3 อันดับแรก พร้อมระยะทางและเส้นทางนำทาง

# ปัญหาที่ต้องการแก้
ผู้ใช้มักหาห้องน้ำที่อยู่ใกล้ได้ยากเมื่ออยู่ในอาคารใหม่หรือพื้นที่ที่ไม่คุ้นเคย ทำให้เสียเวลา ไม่สะดวก และอาจเกิดความเครียด

# กลุ่มผู้ใช้
- นักศึกษา
- บุคลากรในมหาวิทยาลัย
- บุคคลภายนอก / ผู้มาติดต่อ

# ประโยชน์ของระบบ
- ค้นหาห้องน้ำที่ใกล้ที่สุดได้ทันที
- แสดงว่าห้องน้ำฟรีหรือมีค่าใช้จ่าย
- ประหยัดเวลา
- เพิ่มความสะดวกในการเดินทาง

# Input
- ตำแหน่งผู้ใช้ (GPS)
- ปุ่มค้นหาห้องน้ำใกล้ฉัน
- การเลือกห้องน้ำ

# Output
- ห้องน้ำที่ใกล้ที่สุด Top 3
- ระยะทาง
- เส้นทางไปห้องน้ำ

# ฟีเจอร์หลัก
- ค้นหาห้องน้ำใกล้ที่สุด
- แสดงห้องน้ำใกล้ที่สุด 3 อันดับ
- คำนวณระยะทางอัตโนมัติ
- ระบบนำทาง
- แจ้งเตือนเมื่อไม่เปิด GPS

# สถาปัตยกรรมระบบ
- View: Web Application
- Model: SQLite Database
- Controller: GEO Logic และ 3 Distance Selection
- External Integration: Google Maps Routing API

# Libraries ที่ใช้
- `sqlite3`
- `requests`
- `math`

# การพัฒนาต่อในอนาคต
- ปรับปรุงความแม่นยำของ GPS ภายในอาคาร
- รองรับการค้นหาห้องน้ำตามชั้น
- ทดสอบการใช้งานบนมือถือ
- เพิ่มระบบรีวิวจากผู้ใช้
